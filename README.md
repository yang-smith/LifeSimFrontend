# 🌊🌊 人生重开模拟器 🌊🌊

## 概述

**人生重开模拟器** 是一款由 GPT 驱动的模拟人生游戏。

- **版本**: 0.1 beta
- **作者**: 君秋水

## 特点

- **驱动技术**: 利用 GPT 动态生成功能，为玩家提供深度的模拟人生体验。每一次都是全新的体验
- **互动性**: 玩家可以根据自己的选择，体验不同的人生路径和决策。支持pc和手机的网页端
- **更新**: 项目不定期更新。
